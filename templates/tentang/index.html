Tujuan Penelitian
Tujuan Utama

Mengembangkan sistem prediksi harga bahan pangan yang akurat menggunakan Support Vector Regression untuk mendukung monitoring harga dan pengambilan kebijakan di Kabupaten Langkat, Sumatera Utara.

Tujuan Khusus
Menganalisis pola perubahan harga bahan pangan periode 2020-2024
Mengimplementasikan algoritma SVR dengan kernel RBF untuk prediksi harga
Membandingkan performa prediksi pada 6 kategori beras berbeda
Mengevaluasi akurasi model menggunakan metrik MAPE, MAE, dan RMSE
Mengintegrasikan model prediksi ke dalam aplikasi web berbasis Flask
